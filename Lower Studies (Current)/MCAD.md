- Moving Average Convergence Divergence (MACD) 
- Plots the difference between two moving averages. Default is 12EMA and 26EMA.
If 12EMA above 26EMA - MACD positive.
If 12EMA below 26EMA - MACD negative. 

- Trigger line is 9 EMA by default; watch for distance. 

## Four Scenarios To Watch
- Bullish cross over: MACD crosses trigger line to the upside. Indicates buying momentum.
- Bearish cross over: MACD crosses trigger line to the downside. Indicates selling pressure.
- 0 Line crossovers: Bullish cross above 0 line, bearish cross below 0 line.
